# Job Applications BETA-version

Добро пожаловать в репозиторий пет-проекта "Заявки на работу"! Этот проект создан в образовательных целях для изучения технологий JS, React, RTK, MUI material. Давайте рассмотрим основные аспекты этого проекта.

### Описание проекта
Этот пет-проект представляет собой простой набор всех откликов с отображением их статусов, который разработан с использованием следующих технологий и библиотек:

**Tailwind:** CSS-фреймворк, которые можно скомпоновать для создания любого дизайна прямо в вашей разметке.

**JavaScript (JS):** Основной язык программирования для создания интерактивных веб-приложений.

**React:** Библиотека, которая позволяет создавать компоненты и строить пользовательские интерфейсы.

**Redux Toolkit (RTK):** Инструментарий для более удобного управления состоянием приложения.

**MUI material:** Библиотека готовых React компонентов с возможностью кастомизации.

### Установка и запуск
Убедитесь, что у вас установлен Node.js.

**Клонируйте этот репозиторий на свой локальный компьютер:**
```
git clone https://github.com/NewZaikaM/our-command-project
```

**Перейдите в директорию проекта:**
```
cd our-command-project
```

**Установите зависимости:**
```
npm install
```

**Запустите локальный сервер разработки:**
```
npm run dev
```

### Структура проекта
- src/: Исходный код пет-проекта.
  - api/: Эндипоинты для работы с данными из firebase.
  - pages/: Страницы, используемые в приложении.
  - store/: Конфигурация Redux Toolkit, включая состояние, редюсеры и действия.
  - utils/: Упрощающие функции для каких-то действий.
  - router.jsx: Набор маршрутов приложения.
  - index.css: Набор основных стилей.
  - main.jsx: Точка входа в приложение.
